# Sistema de Inventário 📦

Sistema de Gerenciamento de Ativos corporativos/institucionais. Aplicação web full-stack desenvolvida para o controle completo e escalável de patrimônio e equipamentos. 

## 🚀 Tecnologias Utilizadas
* **Backend:** Python 3, Django
* **Banco de Dados:** SQLite (Desenvolvimento) / PostgreSQL (Produção)
* **Gerenciamento de Ambiente:** python-decouple (Variáveis de ambiente)
